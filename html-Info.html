Answers for the html quiz

💻 What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It defines the structure of a web document using tags.

🏷️ What is the correct structure of an HTML document?

                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>My First Web Page</title>
                    </head>
                    <body>
                        <h1>Hello, World!</h1>
                        <p>This is my first webpage.</p>
                    </body>
                    </html>

🏷️ Which HTML element defines the title of a document?

The <title> tag is used to define the name of the web page, which appears in the browser tab.

                    <title>My Awesome Website</title>

🏷️ What is the purpose of the <body> tag?

The <body> tag contains the main content of a webpage.

                    <body>
                        <h1>Welcome!</h1>
                        <p>This is the main content of the webpage.</p>
                    </body>

🏷️ Which HTML tag is used to create a hyperlink?

To create clickable links, use the <a> (anchor) tag.

                    <a href="https://www.google.com">Go to Google</a>

🏷️ Which tag is used to display images in HTML?

To insert an image, use the <img> tag.

                    <img src="image.jpg" alt="Description of the image">

🏷️ What attribute is used to provide the path of an image?

The src attribute specifies the file path or URL of an image.

                    <img src="images/photo.jpg" alt="Mountain view">

🏷️ Which HTML tag is used to create an unordered list?

Use the <ul> (unordered list) tag.

                    <ul>
                        <li>Apple</li>
                        <li>Banana</li>
                        <li>Orange</li>
                    </ul>

🏷️ Ordered Lists

Use the <ol> (ordered list) tag for numbered lists.

                    <ol>
                        <li>Wake up</li>
                        <li>Brush teeth</li>
                        <li>Start work</li>
                    </ol>

🏷️ What does the <br> tag do?

The <br> tag creates a line break.

                    <p>This is line one.<br>This is line two.</p>

🏷️ What does the <fieldset> tag do?

The <fieldset> tag is used to group related elements inside an HTML form.

                    <form>
                        <fieldset>
                            <legend>Personal Information</legend>
                            <label for="name">Name:</label>
                            <input type="text" id="name">
                        </fieldset>
                    </form>
