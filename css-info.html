CSS is a styling language used to control the appearance and layout of HTML elements on a web page. It allows developers to apply colors, fonts, spacing, animations, and responsive designs to create visually appealing websites.

The style attribute is used to define inline styles in HTML.

                    <p style="color: blue; font-size: 16px;">This is a blue text.</p>

üè∑Ô∏è How do you insert a comment in a CSS file?

CSS comments are written inside /* ... */ and are ignored by the browser.

                    /* This is a CSS comment */
                    body {
                        background-color: lightgray;
                    }

üè∑Ô∏è Which property is used to change the background color of an element?

The background-color property is used to set the background color.

                    body {
                        background-color: yellow;
                    }

üè∑Ô∏è How do you apply a style to all <p> elements?

Use the element selector p in CSS.

                    p {
                        font-size: 18px;
                        color: black;
                    }

üè∑Ô∏è Which property is used to change the font of an element?

The font-family property defines the font of an element.

                    body {
                        font-family: Arial, sans-serif;
                    }

üè∑Ô∏è How do you make each word in a text start with a capital letter?

Use the text-transform property with the value capitalize.

                    p {
                        text-transform: capitalize;
                    }

üè∑Ô∏è How do you select an element with the class name "header"?

Use a dot (.) before the class name in CSS.

                    .header {
                        background-color: blue;
                        color: white;
                        padding: 10px;
                    }

üè∑Ô∏è What is the default value of the "position" property?

The default value of the position property is static, meaning elements appear in the normal document flow.

                    div {
                        position: static;
                    }

üè∑Ô∏è What is the purpose of the z-index property in CSS?

The z-index property controls the stack order of overlapping elements. Higher values appear in front of lower values.

                    .box1 {
                        position: absolute;
                        z-index: 2;
                    }
                
                    .box2 {
                        position: absolute;
                        z-index: 1;
                    }

üè∑Ô∏è What does CSS stand for?

CSS (Cascading Style Sheets) is a language used to style HTML elements. It allows you to control the layout, colors, fonts, and positioning of elements on a web page.
