üíª What is JavaScript?

JavaScript (JS) is a high-level, dynamic programming language used primarily for creating interactive web pages. It allows developers to add interactivity, animations, and complex functionality to websites.

üè∑Ô∏è Which syntax is correct to output 'Hello World' in an alert box?

In JavaScript, the alert() function is used to display a pop-up message. To output "Hello World", you can use the following syntax:

                    alert("Hello World");

This will create a pop-up box showing the message "Hello World".

üè∑Ô∏è How do you call a function named 'myFunction'?

To call a function in JavaScript, you simply use the function's name followed by parentheses. In the case of a function named myFunction, you call it like this:

                myFunction();

If the function accepts parameters, you pass the arguments inside the parentheses.

üè∑Ô∏è How to write an IF statement in JavaScript?

An IF statement is used to execute a block of code if a specified condition is true. The basic syntax looks like this:

                if (condition) {
                // Code to execute if condition is true
                }

For example:

                    if (x > 5) {
                    console.log("x is greater than 5");
                    }

üè∑Ô∏è How to write an IF statement for executing some code if 'i' is NOT equal to 5?

To check if a variable i is NOT equal to 5, you use the !== operator. Here's the syntax:

                    if (i !== 5) {
                    // Code to execute if i is NOT equal to 5
                    }

The !== operator checks for inequality (both value and type).

üè∑Ô∏è How does a FOR loop start?

A FOR loop is used to run a block of code a specific number of times. Here's the syntax to start a basic FOR loop:

                    for (let i = 0; i < 10; i++) {
                        // Code to execute
                    }

In this example:

  • let i = 0 initializes the loop counter.
  • i < 10 is the condition that must be true for the loop to continue.
  • i++ increments the counter after each iteration.

üè∑Ô∏è How can you add a single-line comment in JavaScript?

In JavaScript, single-line comments are added by using two forward slashes (//). Everything after // on that line will be ignored by the JavaScript engine. For example:

                    // This is a single-line comment

üè∑Ô∏è What is the correct way to write a JavaScript array?

Arrays in JavaScript are written with square brackets []. You can define an array and populate it with elements separated by commas. Here's the syntax:

                    let fruits = ["apple", "banana", "cherry"];

This defines an array named fruits with three elements.

üè∑Ô∏è How do you find the number with the highest value of x and y?

To find the highest number between two variables x and y, you can use the Math.max() function:

                    let highest = Math.max(x, y);

Math.max() returns the largest of the numbers provided as arguments.

üè∑Ô∏è Which operator is used to assign a value to a variable?

In JavaScript, the assignment operator = is used to assign a value to a variable. Here's an example:

                    let x = 10; // Assigns the value 10 to the variable x

üè∑Ô∏è What is the correct way to write a JavaScript object?

A JavaScript object is written using curly braces {} and consists of key-value pairs. The keys are strings (or symbols), and the values can be any valid JavaScript data type. Here's an example:

                    let person = {
                    name: "John",
                    age: 30,
                    country: "USA"
                    };

In this example, the object person contains three key-value pairs: name, age, and country.
